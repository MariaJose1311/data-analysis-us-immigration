# data-analysis-us-immigration
Análisis Exploratorio de Datos (EDA) del proceso de visas PERM en EEUU. Este proyecto utiliza Python, Pandas y Seaborn para descubrir tendencias en empresas, puestos de trabajo y salarios, transformando datos de Kaggle en insights estratégicos para el sector de la inmigración. Ideal para entender patrones en el mercado laboral de alta cualificación
